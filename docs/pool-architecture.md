# Архитектура пула воркеров

`AccrualWorkerPool` (`internal/order/worker.go`) — фоновый опрос внешней системы расчёта и обновление заказов в БД.

```
                         ┌─────────────────────────────────────────┐
                         │           POST /api/user/orders          │
                         └────────────────────┬────────────────────┘
                                              │
                                              ▼
┌──────────────┐   InsertOrder (NEW)   ┌─────────────┐   Enqueue      ┌──────────────────────────────┐
│  Пользователь │ ───────────────────▶ │ order.Service│ ─────────────▶ │     AccrualWorkerPool        │
└──────────────┘                       └──────┬──────┘                │                              │
                                              │                        │  jobs chan ◀── scanner       │
                                              ▼                        │       │      (poll_interval) │
                                       ┌─────────────┐                │       │      SELECT NEW/     │
                                       │  PostgreSQL │ ◀──────────────│───────┼───   PROCESSING      │
                                       │   orders    │  UpdateAccrual │       ▼                      │
                                       └─────────────┘                │  worker × N                  │
                                              ▲                        │       │                      │
                                              │ CreditAccrual        └───────┼──────────────────────┘
                                              │ (balance, позже)             │
                                       ┌─────────────┐                       │ GET /api/orders/{number}
                                       │   balance   │                       ▼
                                       └─────────────┘              ┌──────────────────┐
                                                                    │ accrualclient    │
                                                                    │ (HTTP)           │
                                                                    └────────┬─────────┘
                                                                             ▼
                                                                    ┌──────────────────┐
                                                                    │ Внешняя система  │
                                                                    │ расчёта начислений│
                                                                    └──────────────────┘
```
